# 数字华容道游戏手册

## 游戏简介
数字华容道是一个经典的益智游戏，玩家需要通过移动数字方块，将打乱的数字按顺序排列。游戏界面简洁直观，操作简单，适合各个年龄段的玩家。

## 游戏规则
1. 游戏开始时，数字1-8和一个空格随机分布在3x3的棋盘上
2. 玩家可以通过点击与空格相邻的数字方块来移动它
3. 移动后的数字会占据空格的位置，原来的位置变成空格
4. 当所有数字按顺序排列（从左到右，从上到下）时，游戏胜利

## 操作说明
### 基本操作
- 点击数字方块：移动选中的数字到空格位置
- 新游戏按钮：开始一局新的游戏
- 重置按钮：重新开始当前游戏
- 撤销按钮：撤销上一步移动（待实现）

### 难度选择
- 在控制面板右侧可以选择游戏难度
- 难度级别1：简单（20步随机移动）
- 难度级别2：中等（50步随机移动）
- 难度级别3：困难（100步随机移动）

## 游戏界面
### 主界面
- 左侧：游戏棋盘，显示数字方块
- 右侧：控制面板，包含游戏信息和操作按钮

### 信息显示
- 步数：显示当前已移动的步数
- 时间：显示游戏进行的时间

### 胜利提示
- 当完成游戏时，会弹出胜利对话框
- 显示总步数和总用时

## 游戏技巧
1. 观察数字的当前位置和目标位置
2. 规划移动路径，避免来回移动
3. 优先移动边缘数字，为中间数字腾出空间
4. 注意保持数字的连续性，避免打乱已排好的数字

## 常见问题
1. 如果点击数字没有反应，请检查：
   - 该数字是否与空格相邻
   - 游戏是否正在进行中
2. 如果游戏无法完成，可以：
   - 点击重置按钮重新开始
   - 选择新的难度级别
   - 开始新游戏
